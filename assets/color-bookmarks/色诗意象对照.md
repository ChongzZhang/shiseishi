## 二、画面补充要求（你曾单独强调）

| 色名 | 额外要求 |
| --- | --- |
| 魏紫 | 菊花疏朗、色调统一；后版保留寒潭秋水，非淡蓝紫 |
| 暮山紫 / 绛紫 | 紫要淡、透明度高，山上仅一点点晕染；后改绛紫为深绛紫全画面 |
| 石榴红 | 顶栏色圆适中；无现代器物 |
| 苍翠 | 图中不加诗句；无外圈白框；标准 1536×1024 供裁剪 |
| 金色 | 晴空；不在水边；不要山 |
| 鹅黄 | 晴空；不在水边；不要山 |
| 彤 | 色值 `#F35336` 须真实体现在半江映红中；顶栏色圆与殷红等书签一致 |

## 三、书签制作约定（系列统一）

- **尺寸**：原图 1536×1024（居中竖书签），裁剪后 **418×1015**
- **顶栏**：米色纸纹 + 色圆 + 色名 + HEX（不加诗句正文）
- **下半部**：水墨淡彩意境画，突出指定色，可有渐变与其他辅色
- **参考版式**：`c92a5_殷红.png`（色圆大小、字号间距）
